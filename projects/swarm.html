Drone Swarm Engineering

We built an experimental flight arena and used it to develop indoor choreographies, as well as a drone-to-simulation framework.

As recorded in my Masters thesis Intelligent Systems for Next Generation Drone Functionality.

    • Hardware Stack: Crazyflie Nanodrones with Optitrack tracking.
    • Tech Stack: Python, C++ and ROS, with swarm modules designed by hand.
    • Github URL: Experimental workspace | As part of the Intelligent Drones Lab
